# 🔍 ShopSpy - AI-помощник покупателя маркетплейсов

Браузерное расширение + локальный бэкенд для умного шопинга на WB и Ozon.

## Что делает

- **Собирает историю цен** - каждый раз когда вы открываете товар, цена записывается в локальную базу
- **Разоблачает фейковые скидки** - анализирует историю и показывает реальную скидку
- **AI-анализ отзывов** - Claude читает отзывы и выдает краткую выжимку: плюсы, минусы, стоит ли покупать
- **Рекомендация** - "покупать / подождать / не покупать" на основе цены и отзывов

## Быстрый старт

### 1. Установите зависимости

```bash
cd shopspy
pip install -r requirements.txt
```

### 2. (Опционально) Настройте Claude API для AI-анализа отзывов

```bash
# Linux / macOS
export ANTHROPIC_API_KEY="sk-ant-..."

# Windows (PowerShell)
$env:ANTHROPIC_API_KEY = "sk-ant-..."

# Windows (cmd)
set ANTHROPIC_API_KEY=sk-ant-...
```

Без API ключа все работает, кроме AI-анализа отзывов.

### 3. Запустите бэкенд

```bash
python backend/main.py
```

Сервер стартует на http://localhost:8000

### 4. Установите расширение в Chrome

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `extension/`

### 5. Используйте

1. Откройте любой товар на [wildberries.ru](https://www.wildberries.ru) или [ozon.ru](https://www.ozon.ru)
2. В правом нижнем углу появится панель ShopSpy
3. Панель покажет анализ цены и кнопку для AI-анализа отзывов
4. Чем больше товаров вы просматриваете, тем больше данных накапливается

## API endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/price` | Записать цену товара |
| GET | `/api/price/history` | История цен товара |
| POST | `/api/reviews/analyze` | AI-анализ отзывов |
| GET | `/api/stats` | Статистика |
| GET | `/api/products` | Список отслеживаемых товаров |
| GET | `/` | Веб-дашборд |
| GET | `/docs` | Swagger документация |

## Иконки расширения

Положите в `extension/icons/`:
- `icon48.png` (48x48)
- `icon128.png` (128x128)

Или расширение будет работать без иконок с дефолтной.

## Ограничения текущей версии

- Данные хранятся только локально на вашем компе
- История цен накапливается только когда вы просматриваете товары
- Селекторы WB/Ozon могут сломаться при обновлении верстки сайтов
- Нет кросс-платформенного сравнения цен (пока)
