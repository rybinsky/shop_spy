# 🔍 ShopSpy — AI-помощник для умных покупок

**ShopSpy** — браузерное расширение с бэкендом для отслеживания цен на Wildberries и Ozon. Помогает экономить, обнаруживать фейковые скидки и получать уведомления о снижении цен в Telegram.

## 🚀 Быстрый старт

### 1. Установка бэкенда

```bash
git clone https://github.com/your-repo/shopspy.git
cd shopspy
pip install -r requirements.txt
```

### 2. Создание Telegram-бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. **Скопируйте токен** — он понадобится для запуска

### 3. Получение Chat ID

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте `/start`
3. **Скопируйте ваш Chat ID**

### 4. Запуск сервера

```bash
# macOS / Linux
TELEGRAM_BOT_TOKEN=ваш_токен python -m backend.main

# Windows (PowerShell)
$env:TELEGRAM_BOT_TOKEN="ваш_токен"
python -m backend.main
```

Сервер запустится на http://localhost:8000

### 5. Установка расширения

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `extension/`

### 6. Авторизация

1. Кликните на иконку расширения ShopSpy
2. Нажмите "Войти через Telegram"
3. Введите ваш Chat ID
4. Готово!

### Сценарии использования

#### Сценарий 1: Просмотр товара (без авторизации)

```
1. Пользователь открывает товар на WB/Ozon
2. Расширение автоматически парсит цены
3. Цены отправляются на сервер (анонимно)
4. Панель показывает:
   - Текущие цены
   - Анализ цены (если есть история)
   - График (если есть ≥2 точки)
   - Ссылка "Найти на другой площадке"
5. Кнопка "Отслеживать" неактивна (требуется авторизация)
```

#### Сценарий 2: Просмотр товара (авторизованный пользователь)

```
1. Пользователь открывает товар на WB/Ozon
2. Расширение парсит цены и отправляет на сервер
3. Дополнительно отправляется событие просмотра (sendView)
4. Панель показывает полную информацию
5. Кнопка "Отслеживать" активна
6. При клике на "Отслеживать":
   - Создаётся alert в базе
   - Telegram бот будет отправлять уведомления при смене цены
```


## 📖 Подробный функционал

### Панель на странице товара

При открытии товара на WB или Ozon справа появляется панель ShopSpy:

#### 1. Блок "Текущие цены"

```
💰 Текущие цены
┌────────────────────────────────────┐
│ 💜 WB Кошелёк: 19 870 ₽  -58%      │
│ 36 026 ₽  ~~48 100 ₽  -25%         │
│ 💰 С кошельком экономия 45%        │
└────────────────────────────────────┘
```

- **Цена по карте/кошельку** — самая низкая цена (фиолетовый/синий цвет)
- **Обычная цена** — цена без карты (белый цвет)
- **Зачёркнутая цена** — "оригинальная" цена до скидки
- **Процент скидки** — рассчитывается автоматически
- **Экономия с картой** — насколько дешевле с картой vs обычная цена

#### 2. Блок "Анализ цены"

```
📊 Анализ цены
┌────────────────────────────────────┐
│ ✅ Хорошая цена!                   │
│ Цена близка к минимуму             │
└────────────────────────────────────┘
```

**Возможные вердикты:**

| Статус | Значение | Когда показывается |
|--------|----------|-------------------|
| ✅ **good_deal** | Хорошая цена, можно брать | Цена ≤ минимальной + 5% |
| 🚨 **fake_discount** | Фейковая скидка | Заявленная скидка большая, но цена выше средней |
| ⚠️ **overpriced** | Цена завышена | Цена близка к максимуму |
| ℹ️ **normal** | Обычная цена | Ничего особенного |
| 📊 **insufficient_data** | Недостаточно данных | Меньше 2 записей в истории |

#### 3. Блок "История цен" (график)

```
📈 История цен (14 точек)
┌────────────────────────────────────┐
│     ╭──────────────────╮          │
│    ╭╯                  ╰─╮       │
│   ╭╯  ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄╮  ╰╮      │
│  ╭╯                   ╰╮  ╰╮     │
│ ╭╯                     ╰╮  ╰╮    │
│╭╯                       ╰╮  ╰╮   │
└────────────────────────────────────┘
мин: 34 500 ₽   сред: 36 000 ₽   макс: 42 000 ₽

━━ Обычная цена
┄┄┄ Цена по карте/кошельку
```

- **Красная линия** — обычная цена
- **Фиолетовая/синяя пунктирная** — цена по карте
- **Серая пунктирная** — средняя цена
- Данные накапливаются при каждом просмотре

#### 4. Блок "Экономия"

```
💸 Вы экономите ~1 500 ₽ по сравнению со средней ценой
```

Показывается только если текущая цена ниже средней.

#### 5. Блок "Отслеживание"

```
🔔 Отслеживание
┌────────────────────────────────────┐
│     [🔔 Отслеживать]               │

```

- Если не авторизован: "Сначала войдите через Telegram"

#### 6. Блок "Сравнить цену"

## 💰 Отслеживание трёх цен

ShopSpy автоматически собирает три типа цен для каждого товара:

### Wildberries

| Цена | Описание | Парсинг |
|------|----------|---------|
| **WB Кошелёк** | Цена с WB Кошельком (самая низкая) | `.priceBlockWalletPrice--RJGuT h2` |
| **Обычная цена** | Цена без WB Кошелька | `.finalPriceWrap--tKHRP h2` |
| **Зачёркнутая** | "Оригинальная" цена до скидки | `.priceBlockOldPrice--qSWAf` |

### Ozon

| Цена | Описание | Парсинг |
|------|----------|---------|
| **Ozon Банк** | Цена по карте Ozon Банка | `.pdp_i1b .tsHeadline600Large` |
| **Обычная цена** | Цена без карты | `.pdp_bj.tsHeadline500Medium` |
| **Зачёркнутая** | "Оригинальная" цена до скидки | `.pdp_i9b.pdp_bj0` |

### Почему это важно?

1. **Реальная экономия** — видите, сколько сэкономите с картой
2. **Детекция манипуляций** — зачёркнутая цена часто "прыгает"
3. **Взвешенное решение** — все данные перед глазами

## ⚙️ Конфигурация

Все настройки в **одном файле** — `config.yaml`. Секреты читаются из `.env`.

### Структура

```
config.yaml     # Все настройки (порты, лимиты, пороги...)
.env            # Только секреты (API ключи, токены)
```

### .env (только секреты)

```bash
TELEGRAM_BOT_TOKEN=your_bot_token
GEMINI_API_KEY=your_gemini_key
ANTHROPIC_API_KEY=your_claude_key    # опционально
ADMIN_SECRET=your_admin_secret       # опционально
```

### config.yaml (пример)

```yaml
# Окружение
environment: development

# Сервер
server:
  host: "0.0.0.0"
  port: 8000

# Анализ цен (пороги для вердиктов)
price_analysis:
  good_deal_threshold: 0.85      # ≤85% от средней = хорошая цена
  overpriced_threshold: 1.10     # ≥110% от средней = дорого
  fake_discount_threshold: 0.30  # скидка ≥30% = проверка на фейк

# Rate Limiting
rate_limit:
  per_ip: 10       # AI-запросов в день с одного IP
  global_limit: 200 # всего AI-запросов в день

# Cleanup
cleanup:
  interval_hours: 24  # запускать каждый день
  keep_days: 365      # хранить данные 1 год
```

### Получение API ключей

**Google Gemini (бесплатно):**
1. [Google AI Studio](https://aistudio.google.com/app/apikey) → Create API Key
2. Добавить в `.env`: `GEMINI_API_KEY=...`

**Anthropic Claude (платно):**
1. [Anthropic Console](https://console.anthropic.com/) → API Keys
2. Добавить в `.env`: `ANTHROPIC_API_KEY=...`

### Таблицы базы данных

#### prices (история цен)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Primary Key |
| `platform` | TEXT | 'wb' или 'ozon' |
| `product_id` | TEXT | ID товара на площадке |
| `product_name` | TEXT | Название |
| `price` | REAL | Цена без карты |
| `original_price` | REAL | Зачёркнутая цена |
| `card_price` | REAL | Цена по карте |
| `url` | TEXT | URL товара |
| `recorded_at` | TIMESTAMP | Время записи |

#### price_alerts (отслеживания)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Primary Key |
| `chat_id` | INTEGER | Telegram Chat ID |
| `platform` | TEXT | Площадка |
| `product_id` | TEXT | ID товара |
| `product_name` | TEXT | Название |
| `target_price` | REAL | Целевая цена (опционально) |
| `last_price` | REAL | Последняя известная цена |
| `url` | TEXT | URL товара |
| `is_active` | BOOLEAN | Активно ли отслеживание |

#### telegram_users (пользователи)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Primary Key |
| `chat_id` | INTEGER | Telegram Chat ID |
| `username` | TEXT | Username в Telegram |
| `first_name` | TEXT | Имя |
| `is_active` | BOOLEAN | Активен ли пользователь |

### Панель разработчика

Доступна по адресу `/admin`. Требует заголовок `X-Admin-Secret` если настроен `ADMIN_SECRET`.

Позволяет:
- Просматривать статистику системы
- Мониторить пользователей
- Просматривать все отслеживания
- видеть последние цены

## 🚢 Деплой на Render.com

### Шаг 1: Создание Web Service

1. Зарегистрируйтесь на [Render](https://render.com)
2. Нажмите **New → Web Service**
3. Подключите GitHub репозиторий

### Шаг 2: Настройка

| Параметр | Значение |
|----------|----------|
| Name | `shopspy` |
| Environment | `Python 3` |
| Build Command | `pip install -r requirements.txt` |
| Start Command | `python -m backend.main` |
| Instance Type | `Free` |

### Шаг 3: Переменные окружения

```
TELEGRAM_BOT_TOKEN=your_bot_token
GEMINI_API_KEY=your_gemini_key
ADMIN_SECRET=your_admin_secret
ENVIRONMENT=production
```

### Шаг 4: Обновление расширения

Измените `extension/config.js`:

```javascript
const SHOPSPY_CONFIG = {
  API_BASE: "https://ваш-приложение.onrender.com",
};
```

### Шаг 5: UptimeRobot (опционально)

На бесплатном тарифе Render "засыпает" через 15 минут неактивности.

1. Зарегистрируйтесь на [UptimeRobot](https://uptimerobot.com)
2. Добавьте монитор для `https://ваш-приложение.onrender.com/health`
3. Интервал: 5 минут

## 🔒 Безопасность

- **Локальное хранение** — все данные в SQLite
- **Нет регистрации** — привязка через Telegram Chat ID
- **Rate limiting** — защита от злоупотребления AI-анализом
- **Admin защита** — `ADMIN_SECRET` для административных endpoints
- **Открытый код** — полный аудит безопасности
